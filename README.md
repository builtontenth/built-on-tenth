# Built on Tenth

Revenue intelligence website for HVAC, roofing, and plumbing companies.

## Stack
- Astro 4 static site
- Vercel hosting
- Astro content collections for articles
- Brevo-powered form endpoints

## Local Development

```bash
npm install
npm run dev
```

Local site: `http://localhost:4321`

## Build

```bash
npm run build
```

Important:
- `dist/` is generated build output
- `node_modules/` is installed locally
- `.astro/` is generated by Astro

These are not source files and should not be committed.

## Content Workflow

Articles live in `src/content/blog/*.md`.

To add a new article:
1. Create a new markdown file in `src/content/blog/`
2. Fill the required frontmatter from `src/content/config.ts`
3. Run `npm run build`

The insights index, article routes, and sitemap are generated from the content collection automatically.
The build now also cleans generated folders and fails if any markdown article is missing from built insight routes, the insights index, or the sitemap.

## Core Routes
- `/` homepage
- `/insights` article hub
- `/insights/[slug]` article template
- `/tools` tools hub
- `/services`
- `/about`
- `/newsletter`
- `/apply`

## Forms

Newsletter and apply flows are wired through server-side endpoints:
- `src/pages/api/subscribe.ts`
- `src/pages/api/apply.ts`

## IndexNow

The repo includes an IndexNow key file and submission script.

```bash
npm run indexnow -- --all
```

Or submit specific URLs:

```bash
npm run indexnow -- /insights/hvac-cost-per-lead-by-channel-2026 /tools/phone-calculator
```
